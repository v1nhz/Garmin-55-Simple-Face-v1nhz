# Garmin Simple Face

Watch face dành cho **Garmin Forerunner 55**.

## 📱 Tính năng hiển thị

Màn hình bao gồm:

- 🕒 Thời gian
- ❤️ Nhịp tim (Heart Rate)
- 👣 Bước chân (Steps)
- 🔥 Calories
- 🔋 Trạng thái pin

## 🛠 Build

Dự án được phát triển bằng **Garmin Connect IQ (Monkey C)**.

Có thể build bằng:

```bash
monkeyc -f monkey.jungle -o simpleface.prg -d fr55
